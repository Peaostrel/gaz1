Il sole bacia le vecchie mura,
mentre il vento sussurra tra i pini.
Un caffè caldo, la strada sicura,
e il mare che brilla nei tuoi confini.

L'ombra si allunga sulla piazza vuota,
batte il rincorrersi di mille passi.
La vita è un canto, una dolce nota,
scritta col gesso sopra i sassi.
